# Prius-Vision

A Project using computer vision to control the steering on a car. 

Hardware used:
Intel UP Board
Arduino UNO
IBT-2 Motor driver
Andymark CIM Motor
Random rotary encoder I found
Intel Realsense R200 Camera

Software used:
Python3
OpenCV
librealsense
pyrealsense

The car being used is a 2005 Toyota Prius
